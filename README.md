## Copies of dotfiles for use in MacOS and Linux

### MacOS Apple Silicon
Install ohmyzsh and plugins (use Brewfile).  
Run setup-macos.sh

### MacOS x86
Install ohmyzsh and plugins (use Brewfile).  
Run setup-macos-x86.sh

### Debian / Ubuntu Linux
Run install-ohmyzsh-deb.sh to install ohmyzsh and required plugins.  
Run setup-debian.sh
